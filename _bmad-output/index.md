### CLI Commands

| Command | Description |
|---------|-------------|
| `canvas todo` | List upcoming assignments |
| `canvas todo --show-all` | Include completed/hidden assignments |
| `canvas exclude <id>` | Exclude an assignment |
| `canvas next-due` | Show time until next due date |
